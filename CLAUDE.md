# CLAUDE.md — demo-schulwebsite

Demo-Repo für die Weiterbildung „Cursor + Git/GitHub — Agentic Workflow". Dient der Live-Demo der Feature-Pipeline, nicht als Produktions-Website.

## Stack

- Statisches HTML/CSS/JS, kein Build-Tool, kein Framework
- Start: `python3 -m http.server` oder `index.html` direkt öffnen

## Konventionen

- Deutsch für Texte, Commit-Messages und Issue-Bodies
- **Markdown zuerst, dann `gh`** — Spezifikation immer in `dokumentation/planung/`, danach Issue anlegen
- Kleine, nachvollziehbare Änderungen — dieses Repo ist bewusst minimal gehalten
